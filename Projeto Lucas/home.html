Principais Tecnologias

HTML

Linguagem de marcação para estruturar e apresentar conteúdo na web, fundamental para o desenvolvimento front-end.

CSS

Linguagem de estilização que permite criar layouts responsivos e designs modernos para websites.

JavaScript

Linguagem de programação versátil que permite criar interatividade e dinamismo em aplicações web.

Node.js

Ambiente de execução JavaScript server-side que permite criar aplicações escaláveis e de alta performance.

Biblioteca JavaScript para construção de interfaces de usuário modernas e reativas com componentes reutilizáveis.

TypeScript

Superset do JavaScript que adiciona tipagem estática e recursos avançados para desenvolvimento mais seguro.
